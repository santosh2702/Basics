# MultiTenant - Database Per Tenant MERN SaaS

A production-ready **Multi-Tenant SaaS Boilerplate** built with **MERN Stack** using **Database-Per-Tenant** architecture.

Each tenant gets its own isolated MongoDB database (`tenant_acme`, `tenant_beta`, etc.) for maximum data isolation and security.
